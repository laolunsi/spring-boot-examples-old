# SpringBoot系列基本教程
该仓库为springboot快速上手教程对应的示例代码，主要讲解SpringBoot相关的技术，后续计划包括SpringSecurity、SpringCloud、Spring源码等内容。

如果该系列教程对您有所帮助的话，还请点个star给予支持！感谢！

---
补充：
> 1. 后缀为demo，表示技术示例、问题解决方案，专注实际问题解决
> 2. 后缀为study，表示对某类问题的深入学习，更关心源码层面
